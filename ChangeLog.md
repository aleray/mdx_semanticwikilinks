Changelog
=========


Version 1.1
-----------

- The make_link function now takes four arguments:
    - `md`
      :    the current Markdown instance
    - `rel`
      :    the value of the rel attribute
    - `target`
      :    the value of the href attribute
    - `label`
      :    the link text, if any given.
